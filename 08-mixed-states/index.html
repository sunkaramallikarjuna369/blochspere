Pure vs Mixed States

Pure states lie on the surface of the Bloch sphere (|r⃗| = 1).

Mixed states lie inside the Bloch sphere (|r⃗| < 1).

What is a Mixed State?

A mixed state represents classical uncertainty about which quantum state the system is in. It's a statistical ensemble of pure states.

The Maximally Mixed State

At the center of the Bloch sphere (r⃗ = 0):

ρ = I/2 = ½[1 0; 0 1]

This represents:

  • Complete ignorance about the qubit state
  • 50-50 mixture of |0⟩ and |1⟩
  • No quantum coherence
  • Minimum purity: Tr(ρ²) = ½

Sources of Mixed States

1. Classical Uncertainty

If we prepare |0⟩ with probability p and |1⟩ with probability (1-p):

ρ = p|0⟩⟨0| + (1-p)|1⟩⟨1|

2. Decoherence

Interaction with environment causes pure states to become mixed.

3. Partial Trace

Tracing out part of an entangled system leaves a mixed state.

Purity and Coherence

The purity Tr(ρ²) measures how "quantum" a state is:

|r⃗|Tr(ρ²)State Type
11Pure (maximum coherence)
0.50.625Partially mixed
00.5Maximally mixed (no coherence)
Tr(ρ²) = ½(1 + |r⃗|²)

Geometric Interpretation

The distance from the center reflects the degree of quantum coherence:

  • Surface (|r⃗| = 1): Maximum coherence, definite superposition
  • Inside (0 < |r⃗| < 1): Partial coherence, some classical uncertainty
  • Center (|r⃗| = 0): No coherence, complete classical mixture

The direction of the Bloch vector reflects the average polarization or dominant component of the state.

Python Example

import numpy as np

def create_mixed_state(p, state1, state2):
    """Create mixed state: p*|ψ1⟩⟨ψ1| + (1-p)*|ψ2⟩⟨ψ2|"""
    rho1 = np.outer(state1, np.conj(state1))
    rho2 = np.outer(state2, np.conj(state2))
    return p * rho1 + (1 - p) * rho2

def purity(rho):
    """Calculate purity Tr(ρ²)"""
    return np.real(np.trace(rho @ rho))

def bloch_vector_length(rho):
    """Calculate |r| from density matrix"""
    sigma_x = np.array([[0, 1], [1, 0]])
    sigma_y = np.array([[0, -1j], [1j, 0]])
    sigma_z = np.array([[1, 0], [0, -1]])
    
    x = np.real(np.trace(rho @ sigma_x))
    y = np.real(np.trace(rho @ sigma_y))
    z = np.real(np.trace(rho @ sigma_z))
    
    return np.sqrt(x**2 + y**2 + z**2)

# Define basis states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

# Create mixed states with different purities
print("Mixed States Analysis")
print("-" * 50)

for p in [1.0, 0.75, 0.5, 0.25, 0.0]:
    rho = create_mixed_state(p, ket_0, ket_1)
    r = bloch_vector_length(rho)
    pur = purity(rho)
    print(f"p = {p:.2f}: |r| = {r:.3f}, Tr(ρ²) = {pur:.3f}")

# Maximally mixed state
print("\nMaximally mixed state (center of Bloch sphere):")
rho_mixed = np.eye(2) / 2
print(f"ρ = I/2:\n{rho_mixed}")
print(f"|r| = {bloch_vector_length(rho_mixed):.3f}")
print(f"Tr(ρ²) = {purity(rho_mixed):.3f}")
