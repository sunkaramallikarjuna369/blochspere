The Density Matrix

Every single-qubit state (pure or mixed) can be represented using a density matrix:

ρ = |ψ⟩⟨ψ| = ½(I + r⃗ · σ⃗)

where:

  • r⃗ = (x, y, z) is the Bloch vector
  • σ⃗ = (σx, σy, σz) are the Pauli matrices
  • I is the 2×2 identity matrix

The Pauli Matrices

σx = [0 1; 1 0]

σy = [0 -i; i 0]

σz = [1 0; 0 -1]

These matrices form a basis for 2×2 Hermitian traceless matrices.

Expanded Form

The density matrix in terms of Bloch coordinates:

ρ = ½ [1+z x-iy; x+iy 1-z]

For a pure state on the surface (|r⃗| = 1):

  • ρ00 = (1 + z)/2 = cos²(θ/2)
  • ρ11 = (1 - z)/2 = sin²(θ/2)
  • ρ01 = (x - iy)/2 = coherence term

Purity

The length of the Bloch vector determines the purity of the quantum state:

Tr(ρ²) = ½(1 + |r⃗|²)
|r⃗|PurityState Type
11Pure state (on surface)
0 < |r⃗| < 1½ < Tr(ρ²) < 1Mixed state (inside)
0½Maximally mixed (center)

The Center

The center of the Bloch sphere (r⃗ = 0) represents the maximally mixed state ρ = I/2, which has no quantum coherence.

Bloch Vector from Density Matrix

Given a density matrix ρ, we can extract the Bloch vector:

x = Tr(ρ σx) = 2 Re(ρ01)
y = Tr(ρ σy) = 2 Im(ρ01)
z = Tr(ρ σz) = ρ00 - ρ11

Python Example

import numpy as np

# Pauli matrices
I = np.eye(2)
sigma_x = np.array([[0, 1], [1, 0]])
sigma_y = np.array([[0, -1j], [1j, 0]])
sigma_z = np.array([[1, 0], [0, -1]])

def bloch_to_density(x, y, z):
    """Convert Bloch vector to density matrix."""
    return 0.5 * (I + x*sigma_x + y*sigma_y + z*sigma_z)

def density_to_bloch(rho):
    """Extract Bloch vector from density matrix."""
    x = np.real(np.trace(rho @ sigma_x))
    y = np.real(np.trace(rho @ sigma_y))
    z = np.real(np.trace(rho @ sigma_z))
    return (x, y, z)

def purity(rho):
    """Calculate purity Tr(ρ²)."""
    return np.real(np.trace(rho @ rho))

# Example: |+⟩ state
x, y, z = 1, 0, 0
rho = bloch_to_density(x, y, z)
print(f"|+⟩ density matrix:\n{rho}")
print(f"Purity: {purity(rho):.4f}")
print(f"Bloch vector: {density_to_bloch(rho)}")

# Example: Maximally mixed state
rho_mixed = bloch_to_density(0, 0, 0)
print(f"\nMaximally mixed state:\n{rho_mixed}")
print(f"Purity: {purity(rho_mixed):.4f}")
