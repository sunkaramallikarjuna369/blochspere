The Big Picture

The Bloch Sphere offers a complete and intuitive visualization of qubit behavior. It connects mathematical formalism with geometric intuition.

Core Insight

Every operation on a qubit — whether a gate, phase shift, or measurement — corresponds to a rotation or projection on this sphere.

State Representation

|ψ⟩ = cos(θ/2)|0⟩ + eiφsin(θ/2)|1⟩

Bloch Vector:

x = sin(θ)cos(φ)
y = sin(θ)sin(φ)
z = cos(θ)

Key Coordinates

State(θ, φ)(x, y, z)
|0⟩(0, 0)(0, 0, 1)
|1⟩(π, 0)(0, 0, -1)
|+⟩(π/2, 0)(1, 0, 0)
|-⟩(π/2, π)(-1, 0, 0)
|+i⟩(π/2, π/2)(0, 1, 0)
|-i⟩(π/2, 3π/2)(0, -1, 0)

Gates as Rotations

GateRotation
X180° about X-axis
Y180° about Y-axis
Z180° about Z-axis
H180° about (X+Z)/√2
S90° about Z-axis
T45° about Z-axis

Measurement

Z-basis:

P(|0⟩) = cos²(θ/2) = (1+z)/2
P(|1⟩) = sin²(θ/2) = (1-z)/2

After measurement: state collapses to |0⟩ or |1⟩ (poles)

Pure vs Mixed

Pure states: |r⃗| = 1 (on surface)

Mixed states: |r⃗| < 1 (inside)

Maximally mixed: r⃗ = 0 (center)

Purity: Tr(ρ²) = ½(1 + |r⃗|²)

Quick Reference Code

import numpy as np

# === BASIC STATES ===
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])
ket_plus = np.array([1, 1]) / np.sqrt(2)
ket_minus = np.array([1, -1]) / np.sqrt(2)

# === PAULI MATRICES ===
I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)

# === BLOCH VECTOR ===
def state_to_bloch(state):
    rho = np.outer(state, np.conj(state))
    x = np.real(np.trace(rho @ X))
    y = np.real(np.trace(rho @ Y))
    z = np.real(np.trace(rho @ Z))
    return (x, y, z)

def bloch_to_state(theta, phi):
    return np.array([np.cos(theta/2), np.exp(1j*phi)*np.sin(theta/2)])

# === MEASUREMENT ===
def measure_prob_0(state):
    return np.abs(state[0])**2

# === PURITY ===
def purity(rho):
    return np.real(np.trace(rho @ rho))
