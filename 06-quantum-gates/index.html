Gates as Rotations

Single-qubit unitary gates act as rotations on the Bloch Sphere:

Rn̂(θ) = e-iθn̂·σ⃗/2

This corresponds to a rotation by angle θ about axis n̂.

Key Insight

Every single-qubit gate is a rotation! The Pauli gates are 180° rotations about their respective axes.

Pauli Gates

Gate Matrix Rotation
X [0 1; 1 0] 180° about X-axis
Y [0 -i; i 0] 180° about Y-axis
Z [1 0; 0 -1] 180° about Z-axis

Hadamard Gate

H = (X + Z)/√2 = [1 1; 1 -1]/√2

The Hadamard gate rotates 180° about the axis (X + Z)/√2, which is the diagonal between X and Z axes.

Effect:

  • H|0⟩ = |+⟩ (North pole → +X)
  • H|1⟩ = |-⟩ (South pole → -X)
  • H|+⟩ = |0⟩ (+X → North pole)

Phase Gates

Gate Matrix Rotation
S [1 0; 0 i] 90° about Z-axis
T [1 0; 0 eiπ/4] 45° about Z-axis

Phase gates only affect the relative phase (longitude), not the amplitude (latitude).

Python Example

import numpy as np

# Define gates
I = np.eye(2)
X = np.array([[0, 1], [1, 0]])
Y = np.array([[0, -1j], [1j, 0]])
Z = np.array([[1, 0], [0, -1]])
H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
S = np.array([[1, 0], [0, 1j]])
T = np.array([[1, 0], [0, np.exp(1j * np.pi / 4)]])

# Define states
ket_0 = np.array([1, 0])
ket_1 = np.array([0, 1])

def state_to_bloch(state):
    """Convert state to Bloch coordinates."""
    rho = np.outer(state, np.conj(state))
    x = 2 * np.real(rho[0, 1])
    y = 2 * np.imag(rho[0, 1])
    z = np.real(rho[0, 0] - rho[1, 1])
    return (x, y, z)

# Demonstrate gate actions
print("Gate actions on |0⟩:")
print(f"Initial |0⟩: {state_to_bloch(ket_0)}")
print(f"X|0⟩ = |1⟩: {state_to_bloch(X @ ket_0)}")
print(f"H|0⟩ = |+⟩: {state_to_bloch(H @ ket_0)}")
print(f"S|+⟩:       {state_to_bloch(S @ H @ ket_0)}")

# Verify rotations
print("\nVerify: X is 180° rotation about X-axis")
print(f"|0⟩ → X → |1⟩: (0,0,1) → (0,0,-1)")
print(f"Actual: {state_to_bloch(ket_0)} → {state_to_bloch(X @ ket_0)}")
