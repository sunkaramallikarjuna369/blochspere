Exercise 1: Bloch Vector Calculation

Problem: Derive the Bloch vector (x, y, z) for the state:

|ψ⟩ = cos(π/6)|0⟩ + eiπ/4sin(π/6)|1⟩

Solution:

From the general form |ψ⟩ = cos(θ/2)|0⟩ + eiφsin(θ/2)|1⟩:

  • θ/2 = π/6, so θ = π/3
  • φ = π/4

The Bloch vector coordinates are:

x = sin(θ)cos(φ) = sin(π/3)cos(π/4) = (√3/2)(√2/2) = √6/4 ≈ 0.612
y = sin(θ)sin(φ) = sin(π/3)sin(π/4) = (√3/2)(√2/2) = √6/4 ≈ 0.612
z = cos(θ) = cos(π/3) = 1/2 = 0.5

Answer: r⃗ = (√6/4, √6/4, 1/2) ≈ (0.612, 0.612, 0.5)

Exercise 2: Gate Rotation

Problem: Show that Rz(π/2) rotates the state |+⟩ to |+i⟩.

Solution:

The Rz(π/2) gate is:

Rz(π/2) = [e-iπ/4 0; 0 eiπ/4]

Applying to |+⟩ = (|0⟩ + |1⟩)/√2:

Rz(π/2)|+⟩ = (e-iπ/4|0⟩ + eiπ/4|1⟩)/√2

Factor out the global phase e-iπ/4:

= e-iπ/4(|0⟩ + eiπ/2|1⟩)/√2 = e-iπ/4(|0⟩ + i|1⟩)/√2

Since global phase doesn't matter: this is |+i⟩!

Geometrically: |+⟩ is at (+X axis), rotating 90° about Z gives (+Y axis) = |+i⟩

Exercise 3: Density Matrix

Problem: Compute the density matrix of |-⟩ = (|0⟩ - |1⟩)/√2 and find its Bloch vector.

Solution:

The state vector is:

|-⟩ = [1/√2, -1/√2]T

The density matrix ρ = |−⟩⟨−|:

ρ = [1/√2; -1/√2] × [1/√2, -1/√2]
= [1/2, -1/2; -1/2, 1/2]

Extract Bloch vector using ρ = ½(I + r⃗·σ⃗):

x = 2·Re(ρ₀₁) = 2·(-1/2) = -1
y = 2·Im(ρ₀₁) = 0
z = ρ₀₀ - ρ₁₁ = 1/2 - 1/2 = 0

Answer: r⃗ = (-1, 0, 0), which is the -X axis as expected!

Exercise 4: Sequential Gates

Problem: Visualize how the Hadamard and Z gates act sequentially on |0⟩ on the Bloch sphere.

Solution:

Step 1: Start at |0⟩ = (0, 0, 1) — North pole

Step 2: Apply H (180° rotation about (X+Z)/√2 axis):

H|0⟩ = |+⟩ = (1, 0, 0) — +X axis

Step 3: Apply Z (180° rotation about Z axis):

Z|+⟩ = |-⟩ = (-1, 0, 0) — -X axis

Path on Bloch sphere:

North pole → +X axis → -X axis

The Z gate flips the sign of the |1⟩ component, which on the equator means moving to the opposite side.

Exercise 5: Global Phase

Problem: Explain why a global phase factor eiγ does not change the position on the Bloch sphere.

Solution:

Consider a state |ψ⟩ and its globally-phased version eiγ|ψ⟩.

The density matrix for eiγ|ψ⟩ is:

ρ' = (eiγ|ψ⟩)(e-iγ⟨ψ|) = eiγe-iγ|ψ⟩⟨ψ| = |ψ⟩⟨ψ| = ρ

Since the density matrix is unchanged, and the Bloch vector is derived from the density matrix:

x = Tr(ρσx), y = Tr(ρσy), z = Tr(ρσz)

The Bloch vector (x, y, z) remains the same!

Physical interpretation: Global phase has no observable consequences — all measurement probabilities and expectation values are identical.

Exercise 6: Measurement Probability

Problem: A qubit is in state |ψ⟩ with Bloch vector (0.6, 0, 0.8). What is the probability of measuring |0⟩?

Solution:

For Z-basis measurement, the probability of |0⟩ depends on the z-coordinate:

P(|0⟩) = (1 + z)/2 = (1 + 0.8)/2 = 0.9 = 90%

Alternatively, from z = cos(θ):

θ = arccos(0.8) ≈ 0.6435 rad
P(|0⟩) = cos²(θ/2) = cos²(0.3218) ≈ 0.9 = 90%

Answer: 90% probability of measuring |0⟩
