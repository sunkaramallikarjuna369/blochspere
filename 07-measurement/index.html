Measurement as Projection

Quantum measurement corresponds to projecting the qubit state onto one of the measurement basis axes.

P(0) = cos²(θ/2)
P(1) = sin²(θ/2)

After measurement, the Bloch vector jumps to the north or south pole (for Z-basis measurement).

State Collapse

Before measurement: superposition state anywhere on sphere

After measurement: collapsed to one of the basis states

Irreversible Process

Measurement destroys the superposition. The original state cannot be recovered!

Z-Basis Measurement

The standard computational basis measurement:

  • Projects onto the Z-axis
  • Outcomes: |0⟩ (north pole) or |1⟩ (south pole)
  • Probability depends on "latitude" (θ)
|0⟩: z = +1 (north pole)
|1⟩: z = -1 (south pole)

X-Basis Measurement

Measuring in the Hadamard basis:

  • Projects onto the X-axis
  • Outcomes: |+⟩ (+X) or |-⟩ (-X)
  • Probability depends on x-coordinate
P(+) = (1 + x)/2
P(-) = (1 - x)/2

Y-Basis Measurement

Measuring in the circular polarization basis:

  • Projects onto the Y-axis
  • Outcomes: |+i⟩ (+Y) or |-i⟩ (-Y)
  • Probability depends on y-coordinate
P(+i) = (1 + y)/2
P(-i) = (1 - y)/2

Geometric Interpretation

For any measurement basis:

  1. Draw a line from the state point perpendicular to the measurement axis
  2. The projection point determines the probabilities
  3. After measurement, state jumps to one of the axis endpoints

Python Example

import numpy as np

def measure_z_basis(state, num_shots=1000):
    """Simulate Z-basis measurement."""
    p0 = np.abs(state[0])**2
    p1 = np.abs(state[1])**2
    
    outcomes = np.random.choice([0, 1], size=num_shots, p=[p0, p1])
    return outcomes

def measure_x_basis(state, num_shots=1000):
    """Simulate X-basis measurement."""
    # Transform to X basis
    H = np.array([[1, 1], [1, -1]]) / np.sqrt(2)
    state_x = H @ state
    
    p_plus = np.abs(state_x[0])**2
    p_minus = np.abs(state_x[1])**2
    
    outcomes = np.random.choice(['+', '-'], size=num_shots, p=[p_plus, p_minus])
    return outcomes

# Create a state at θ = π/3
theta = np.pi / 3
state = np.array([np.cos(theta/2), np.sin(theta/2)])

print(f"State: cos({theta/np.pi:.2f}π/2)|0⟩ + sin({theta/np.pi:.2f}π/2)|1⟩")
print(f"P(|0⟩) = {np.abs(state[0])**2:.4f}")
print(f"P(|1⟩) = {np.abs(state[1])**2:.4f}")

# Simulate measurements
z_outcomes = measure_z_basis(state, 10000)
print(f"\nZ-basis measurement (10000 shots):")
print(f"  |0⟩: {np.sum(z_outcomes == 0)} ({np.mean(z_outcomes == 0)*100:.1f}%)")
print(f"  |1⟩: {np.sum(z_outcomes == 1)} ({np.mean(z_outcomes == 1)*100:.1f}%)")
